# mlx-capivara

Uma simples capivara no seu desktop linux <br>
(fork baseado no projeto original do Jérémy https://github.com/jerem-ma/goose-linux)

### Tecnologias

- [ANSI C ](https://www.ansi.org/) 
- [Minilibx](https://github.com/42Paris/minilibx-linux) 


## Instalação
```bash
make
```

## Utilização
```bash
./capivara
```
